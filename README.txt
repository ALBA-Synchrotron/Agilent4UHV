
This device server is part of the ALBA Vacuum suite and requires Serial, 
VacuumController and Fandango modules available in your PYTHONPATH

Modules can be obtained from:

  https://svn.code.sf.net/p/tango-ds/code/DeviceClasses/Vacuum/VacuumController
  
  https://svn.code.sf.net/p/tango-ds/code/DeviceClasses/Communication/Serial

  https://github.com/tango-controls/fandango


For more information check the VacuumController/CHANGES and README.txt files

----

Testing the device server

The communication parameters are:

baudrate 9600
charlength 8
newline 13
parity -
stopbits 1
timeout 500
